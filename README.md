<h1>Tube Companion :o</h1>

`This is a basic app made to check various aspects of the London Underground
as of now the features that it provides are very minimal
(a grand total of one...)`
<p>
  
  `The TUI of the app was made possible with
Textual. You can find the repository for it` [here](https://github.com/Textualize/textual)

</p>
<p>
  
  `Thanks to Transport For London for creating the Unified API
  (where it fetches the data)`

</p>

<h2>How to run</h2>
`This assumes that you already have python installed on your system.`
`If not you can get the latest version` [here](https://www.python.org/downloads/)
<p>1) Extract the zip and open a command prompt in the folder.</p>
<p>2) Install the required libraries.</p>

`py -m pip install -r requirements.txt` OR `pip install -r requirements.txt`

<p>3) <b>OPTIONAL</b>: Add your TFL API key to 'config.py'</p>
<p>4) Run 'maingui.py'</p>
